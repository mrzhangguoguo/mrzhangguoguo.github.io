## 自动化测试脚本

### 快速验证脚本
```bash
# 运行所有基础测试
./test/run_tests.sh
```

### 单项测试命令
```bash
# 测试构建
bundle exec jekyll build

# 测试服务器
bundle exec jekyll serve

# 验证 HTML 有效性
bundle exec htmlproofer ./_site

# 检查 CSS 有效性
stylelint assets/css/*.css
```
